# App Icons

This directory should contain app icons in various sizes for PWA and mobile apps.

## Required Icon Sizes:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## How to Generate Icons:

### Option 1: Online Tool (Easiest)
1. Go to: https://www.pwabuilder.com/imageGenerator
2. Upload your logo (1024x1024px recommended)
3. Click "Generate"
4. Download the zip file
5. Extract all PNG files to this directory

### Option 2: Android Asset Studio
1. Go to: https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html
2. Upload your logo
3. Customize colors and padding
4. Download and extract to this directory

### Option 3: Manual Creation
Use any image editor (Photoshop, GIMP, etc.) to resize your logo to each size listed above.

## Quick Start Icons

For quick testing, you can use a simple colored square:

1. Open MS Paint or any image editor
2. Create a 512x512px image
3. Fill with green color (#4CAF50)
4. Add white text "DS" in the center
5. Save as PNG
6. Use online tool to generate all sizes
