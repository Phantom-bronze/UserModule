================================================================================
STEP 3: ADD DATABASE
================================================================================
[ ] Click "+ New" button
[ ] Select "Database"
[ ] Choose "Add PostgreSQL"
[ ] Wait ~30 seconds

================================================================================
STEP 4: GENERATE SECURE KEYS
================================================================================
Run these commands to generate keys:

python -c "import secrets; print('SECRET_KEY=' + secrets.token_hex(32))"
python -c "import secrets; print('ENCRYPTION_KEY=' + secrets.token_hex(32))"

[ ] Copy SECRET_KEY
[ ] Copy ENCRYPTION_KEY
[ ] Save both for next step

================================================================================
STEP 5: SET ENVIRONMENT VARIABLES
================================================================================
[ ] Click on your web service (not database)
[ ] Go to "Variables" tab
[ ] Click "Raw Editor"
[ ] Paste this template (update the values!):

--------------------------------------------------------------------------------
APP_ENV=production
APP_NAME=Simple Digital Signage
APP_VERSION=1.0.0
DEBUG=False
HOST=0.0.0.0

SECRET_KEY=<paste-your-generated-key>
ENCRYPTION_KEY=<paste-your-generated-key>
ALGORITHM=HS256

ACCESS_TOKEN_EXPIRE_MINUTES=30
REFRESH_TOKEN_EXPIRE_DAYS=7
INVITATION_TOKEN_EXPIRE_HOURS=72

GOOGLE_CLIENT_ID=<your-client-id>.apps.googleusercontent.com
GOOGLE_CLIENT_SECRET=<your-client-secret>
GOOGLE_SCOPES=openid,https://www.googleapis.com/auth/userinfo.email,https://www.googleapis.com/auth/userinfo.profile

CORS_ORIGINS=*
CORS_ALLOW_CREDENTIALS=True

LOG_LEVEL=INFO
LOG_FORMAT=json
--------------------------------------------------------------------------------

[ ] Update SECRET_KEY with generated key
[ ] Update ENCRYPTION_KEY with generated key
[ ] Update GOOGLE_CLIENT_ID
[ ] Update GOOGLE_CLIENT_SECRET
[ ] Click "Save"

================================================================================
[ ] Go to: https://console.cloud.google.com
[ ] Navigate to: APIs & Services ‚Üí Credentials

STEP 8: UPDATE RAILWAY REDIRECT URI
================================================================================

üîß TROUBLESHOOTING
================================================================================

If something doesn't work:

1. Check deployment logs:
   Railway ‚Üí Your Service ‚Üí Deployments ‚Üí View Logs

2. Verify environment variables:
   Railway ‚Üí Your Service ‚Üí Variables ‚Üí Check all are set

3. Check database connection:
   Railway ‚Üí Make sure PostgreSQL is in same project

4. Common issues:
   - SECRET_KEY not set ‚Üí Generate and add
   - GOOGLE_CLIENT_ID wrong ‚Üí Copy from Google Console
   - Redirect URI mismatch ‚Üí Must match exactly
